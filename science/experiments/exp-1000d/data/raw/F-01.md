Not because credentials were weak — because they were the wrong unit entirely. Turns out they are.

"You are a senior software engineer with 10 years of experience" and "You are a senior backend engineer who checks whether the write is fenced, whether the TTL arithmetic holds under GC pause, whether the commit boundary is where the failure mode actually lives" carry the same seniority, the same compulsion framing, the same everything — and produce dramatically different outputs.

Same credential. Different consideration set.

The model doesn't need to know it's an expert. It needs to arrive at the first task token with a semantic neighborhood that already contains the failure modes it's supposed to find. "Senior software engineer" sits at the intersection of web development, systems programming, deployment, testing — every domain pulls, none dominates. The attention is distributed. It spreads thin and stays thin. "TTL arithmetic under GC pause" is a gravity well. The model searches inside it.

That's the mechanism. Domain vocabulary adjacent to the identity anchor pre-weights attention before the task is read. Dense, mechanism-level vocabulary — tokens that name the causal chain, not the category — shapes the consideration set before the task loads. "Production readiness" tells the model where to look. "Whether two workers can both believe they hold the same lock" tells the model what the failure looks like when it gets there. The difference is a question of resolution. One gives the model a zip code. The other gives it the room, the wiring, the load-bearing wall.

The compulsion framing — "cannot help but," "constitutionally unable to" — isn't doing what most practitioners think it's doing.

The signal was in experiment four. Filed under noise.

Strip the domain vocabulary, keep the compulsion: output collapses to baseline. The framing hangs in the air with nothing to attach to. Strip the compulsion, keep the domain vocabulary: same results as the compulsion version, or better.

Every strong persona you've ever written that worked? The domain vocabulary did it. The compulsion came along for the ride.

The practical rule is simple: write the technical substrate, not the credential. Entangle the behavioral drive and the domain vocabulary in the same sentence, not consecutive ones. Name the mechanism, not the orientation.
