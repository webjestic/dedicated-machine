# Production Rate Limiter in Node.js

Let me design this fully before writing a line of code.

---

## Phase 1: Requirements Design

### What algorithm?

| Algorithm | Pros | Cons | Best For |
|---|---|---|---|
| Fixed Window | Simple, fast | Boundary burst problem | Internal tools |
| Sliding Window Log | Precise | Memory heavy (stores every request) | Low-volume, high-precision |
| Sliding Window Counter | Precise + memory efficient | Slight approximation | **General APIs** |
| Token Bucket | Smooth bursting | State complexity | Streaming, SDKs |
| Leaky Bucket | Strict output rate | No burst tolerance | Payment processing |

**Decision: Sliding Window Counter** — handles the boundary burst problem that kills fixed window implementations in production, memory efficient, and the approximation error is bounded and acceptable.

### What are we rate limiting on?
- Per IP (unauthenticated traffic)
- Per API key (authenticated traffic)
- Per user ID (authenticated sessions)
- Composite keys (IP + endpoint, user + endpoint)

### What backing store?

**Redis** — the only correct answer for production:
- Atomic operations via Lua scripts (prevents race conditions)
- Sub-millisecond latency
- TTL-based key expiration (no manual cleanup)
- Survives Node.js process restarts
- Horizontal scaling works correctly

**In-memory fallback** — for when Redis is unavailable (fail-open vs fail-closed is a business decision I'll make explicit)

---

## Phase 2: Sliding Window Counter — The Math

```
Window size: 60 seconds
Current time: 75 seconds

Previous window (0-60s):   count = 40 requests
Current window (60-120s):  count = 12 requests
Time into current window:  15s (25% through)

Weighted count = (40 × 0.75) + 12 = 42 requests
```

This approximation error is at most `(limit / window_size)` requests — acceptable for all standard use cases.

### `scripts/lua/slidingWindow.lua`

```lua
-- Sliding Window Counter
-- KEYS[1] = current window key
-- KEYS[2] = previous window key  
-- ARGV[1] = current window timestamp (seconds)
-- ARGV[2] = window size (seconds)
-- ARGV[3] = limit
-- ARGV[4] = current timestamp (milliseconds, for precision)

local current_key = KEYS[1]
local previous_key = KEYS[2]
local window_size = tonumber(ARGV[2])
local limit = tonumber(ARGV[3])
local now_ms = tonumber(ARGV[4])
local window_start_ms = tonumber(ARGV[1]) * 1000

-- How far into the current window are we? (0.0 to 1.0)
local elapsed_fraction = (now_ms - window_start_ms) / (window_size * 1000)
local previous_weight = 1.0 - elapsed_fraction

-- Get counts
local current_count = tonumber(redis.call('GET', current_key)) or 0
local previous_count = tonumber(redis.call('GET', previous_key)) or 0

-- Weighted estimate of requests in the rolling window
local weighted_count = (previous_count * previous_weight) + current_count

if weighted_count >= limit then
  -- Denied — return current state without incrementing
  return {
    0,                              -- allowed: false
    math.floor(weighted_count),    -- current count
    current_count,                 -- raw current window count
    previous_count                 -- raw previous window count
  }
end

-- Allowed — increment current window
local new_count = redis.call('INCR', current_key)

-- Set TTL on first write to this key (2 windows to cover previous window reads)
if new_count == 1 then
  redis.call('EXPIRE', current_key, window_size * 2)
end

weighted_count = (previous_count * previous_weight) + new_count

return {
  1,                              -- allowed: true
  math.floor(weighted_count),    -- current weighted count
  new_count,                     -- raw current window count
  previous_count                 -- raw previous window count
}
```

### `src/store/redisStore.js`

```javascript
'use strict';

const { createClient } = require('redis');
const fs = require('fs');
const path = require('path');
const crypto = require('crypto');

const LUA_SCRIPT = fs.readFileSync(
  path.join(__dirname, '../../scripts/lua/slidingWindow.lua'),
  'utf8'
);

class RedisStore {
  constructor(options = {}) {
    this.options = {
      url: options.url || process.env.REDIS_URL || 'redis://localhost:6379',
      socket: {
        connectTimeout: options.connectTimeout || 5000,
        // Reconnect with exponential backoff, capped at 30s
        reconnectStrategy: (retries) => {
          if (retries > 10) return new Error('Redis max retries exceeded');
          return Math.min(retries * 100, 30000);
        },
      },
      ...options.redisOptions,
    };

    this.client = null;
    this.scriptSha = null;
    this.isConnected = false;
    this.isReady = false;
  }

  async connect() {
    this.client = createClient(this.options);

    this.client.on('connect', () => {
      this.isConnected = true;
      console.info('[RateLimiter] Redis connected');
    });

    this.client.on('ready', async () => {
      this.isReady = true;
      // Load Lua script into Redis script cache
      await this._loadScript();
      console.info('[RateLimiter] Redis ready, Lua script loaded');
    });

    this.client.on('error', (err) => {
      // Log but don't throw — handled by fallback
      console.error('[RateLimiter] Redis error:', err.message);
    });

    this.client.on('disconnect', () => {
      this.isConnected = false;
      this.isReady = false;
      console.warn('[RateLimiter] Redis disconnected');
    });

    this.client.on('reconnecting', () => {
      console.warn('[RateLimiter] Redis reconnecting...');
    });

    await this.client.connect();
  }

  async _loadScript() {
    try {
      // SCRIPT LOAD returns the SHA1 hash of the script
      // Subsequent calls use EVALSHA — avoids re-sending script bytes
      this.scriptSha = await this.client.scriptLoad(LUA_SCRIPT);
    } catch (err) {
      console.error('[RateLimiter] Failed to load Lua script:', err.message);
      this.scriptSha = null;
    }
  }

  /**
   * Execute the sliding window check atomically.
   * Returns null if Redis is unavailable (caller handles fallback).
   */
  async slidingWindowIncrement({ key, windowSizeSeconds, limit, nowMs }) {
    if (!this.isReady || !this.client) {
      return null;
    }

    const nowSeconds = Math.floor(nowMs / 1000);
    // Align to window boundary
    const currentWindowStart = nowSeconds - (nowSeconds % windowSizeSeconds);
    const previousWindowStart = currentWindowStart - windowSizeSeconds;

    const currentKey = `${key}:${currentWindowStart}`;
    const previousKey = `${key}:${previousWindowStart}`;

    try {
      let result;

      if (this.scriptSha) {
        try {
          result = await this.client.evalSha(this.scriptSha, {
            keys: [currentKey, previousKey],
            arguments: [
              String(currentWindowStart),
              String(windowSizeSeconds),
              String(limit),
              String(nowMs),
            ],
          });
        } catch (err) {
          // NOSCRIPT error means Redis flushed script cache (restart/failover)
          if (err.message && err.message.includes('NOSCRIPT')) {
            await this._loadScript();
            // Retry once with fresh SHA
            result = await this.client.evalSha(this.scriptSha, {
              keys: [currentKey, previousKey],
              arguments: [
                String(currentWindowStart),
                String(windowSizeSeconds),
                String(limit),
                String(nowMs),
              ],
            });
          } else {
            throw err;
          }
        }
      } else {
        // SHA not available, fall back to inline EVAL
        result = await this.client.eval(LUA_SCRIPT, {
          keys: [currentKey, previousKey],
          arguments: [
            String(currentWindowStart),
            String(windowSizeSeconds),
            String(limit),
            String(nowMs),
          ],
        });
      }

      const [allowed, weightedCount, currentCount, previousCount] = result;

      return {
        allowed: allowed === 1,
        weightedCount: Number(weightedCount),
        currentCount: Number(currentCount),
        previousCount: Number(previousCount),
        currentWindowStart,
        windowSizeSeconds,
      };
    } catch (err) {
      console.error('[RateLimiter] Redis eval error:', err.message);
      return null; // Signal fallback
    }
  }

  async disconnect() {
    if (this.client) {
      await this.client.disconnect();
    }
  }

  isHealthy() {
    return this.isConnected && this.isReady;
  }
}

module.exports = RedisStore;
```

### `src/algorithms/inMemoryFallback.js`

```javascript
'use strict';

/**
 * In-memory sliding window counter.
 * Used when Redis is unavailable.
 *
 * OPERATIONAL NOTE: This is per-process only.
 * In a multi-process/multi-instance deployment, each process
 * has its own counters — effective limit becomes:
 *   actual_limit = configured_limit × number_of_instances
 *
 * This is intentional fail-open behavior. If you need fail-closed,
 * set fallbackBehavior: 'deny' in config instead.
 */
class InMemoryFallback {
  constructor() {
    // Map of key -> { windows: Map<windowStart, count>, lastAccess: timestamp }
    this.store = new Map();
    // Cleanup every 5 minutes to prevent unbounded memory growth
    this._cleanupInterval = setInterval(() => this._cleanup(), 5 * 60 * 1000);
    // Don't hold the process open for cleanup
    this._cleanupInterval.unref();
  }

  slidingWindowIncrement({ key, windowSizeSeconds, limit, nowMs }) {
    const nowSeconds = Math.floor(nowMs / 1000);
    const currentWindowStart = nowSeconds - (nowSeconds % windowSizeSeconds);
    const previousWindowStart = currentWindowStart - windowSizeSeconds;

    if (!this.store.has(key)) {
      this.store.set(key, { windows: new Map(), lastAccess: nowMs });
    }

    const entry = this.store.get(key);
    entry.lastAccess = nowMs;

    const currentCount = entry.windows.get(currentWindowStart) || 0;
    const previousCount = entry.windows.get(previousWindowStart) || 0;

    const elapsedFraction =
      (nowMs - currentWindowStart * 1000) / (windowSizeSeconds * 1000);
    const previousWeight = 1.0 - elapsedFraction;
    const weightedCount = previousCount * previousWeight + currentCount;

    if (weightedCount >= limit) {
      return {
        allowed: false,
        weightedCount: Math.floor(weightedCount),
        currentCount,
        previousCount,
        currentWindowStart,
        windowSizeSeconds,
      };
    }

    const newCount = currentCount + 1;
    entry.windows.set(currentWindowStart, newCount);

    // Prune old windows for this key (keep only current and previous)
    for (const [windowStart] of entry.windows) {
      if (windowStart < previousWindowStart) {
        entry.windows.delete(windowStart);
      }
    }

    return {
      allowed: true,
      weightedCount: Math.floor(previousCount * previousWeight + newCount),
      currentCount: newCount,
      previousCount,
      currentWindowStart,
      windowSizeSeconds,
    };
  }

  _cleanup() {
    const now = Date.now();
    const maxIdleMs = 10 * 60 * 1000; // 10 minutes
    let cleaned = 0;

    for (const [key, entry] of this.store) {
      if (now - entry.lastAccess > maxIdleMs) {
        this.store.delete(key);
        cleaned++;
      }
    }

    if (cleaned > 0) {
      console.info(`[RateLimiter] In-memory fallback: cleaned ${cleaned} stale keys`);
    }
  }

  destroy() {
    clearInterval(this._cleanupInterval);
    this.store.clear();
  }
}

module.exports = InMemoryFallback;
```

### `src/utils/keyResolver.js`

```javascript
'use strict';

const crypto = require('crypto');

/**
 * Extracts and normalizes the rate limit key from a request.
 * Order of precedence: API key > User ID > IP
 */

function resolveKey(req, options = {}) {
  const { keyPrefix = 'rl', keyBy = 'auto' } = options;

  let identifier;
  let identifierType;

  if (keyBy === 'ip' || keyBy === 'auto') {
    const ip = extractIP(req);
    identifier = ip;
    identifierType = 'ip';
  }

  if (keyBy === 'apiKey' || keyBy === 'auto') {
    const apiKey = extractAPIKey(req);
    if (apiKey) {
      identifier = apiKey;
      identifierType = 'apikey';
    }
  }

  if (keyBy === 'userId' || keyBy === 'auto') {
    const userId = extractUserId(req);
    if (userId) {
      identifier = userId;
      identifierType = 'user';
    }
  }

  if (keyBy === 'custom' && typeof options.keyGenerator === 'function') {
    identifier = options.keyGenerator(req);
    identifierType = 'custom';
  }

  if (!identifier) {
    // Last resort — hash the user-agent + IP to get something stable
    const raw = `${extractIP(req)}:${req.headers['user-agent'] || 'unknown'}`;
    identifier = crypto.createHash('sha256').update(raw).digest('hex').slice(0, 16);
    identifierType = 'fallback';
  }

  // Include endpoint in key if per-route limiting is enabled
  const routeSegment = options.perRoute
    ? `:${req.method}:${normalizeRoute(req.path || req.url)}`
    : '';

  return {
    key: `${keyPrefix}:${identifierType}:${identifier}${routeSegment}`,
    identifierType,
    identifier,
  };
}

function extractIP(req) {
  // Trust proxy headers only if explicitly configured
  // SECURITY: Never trust X-Forwarded-For without validating proxy chain
  const forwarded = req.headers['x-forwarded-for'];
  if (forwarded && req.app && req.app.get('trust proxy')) {
    // Take the first IP (client), not the last (last proxy)
    return forwarded.split(',')[0].trim();
  }
  return (
    req.ip ||
    req.connection?.remoteAddress ||
    req.socket?.remoteAddress ||
    '0.0.0.0'
  );
}

function extractAPIKey(req) {
  // Check Authorization header: "Bearer <key>" or "ApiKey <key>"
  const auth = req.headers['authorization'];
  if (auth) {
    const match = auth.match(/^(?:Bearer|ApiKey)\s+(.+)$/i);
    if (match) return hashKey(match[1]);
  }
  // Check X-API-Key header
  if (req.headers['x-api-key']) {
    return hashKey(req.headers['x-api-key']);
  }
  // Check query param (not recommended but common)
  if (req.query && req.query.api_key) {
    return hashKey(req.query.api_key);
  }
  return null;
}

function extractUserId(req) {
  // Assumes auth middleware has already run and set req.user
  return req.user?.id || req.user?.sub || null;
}

function hashKey(rawKey) {
  // Never store raw API keys in Redis — hash them
  return crypto.createHash('sha256').update(rawKey).digest('hex').slice(0, 32);
}

function normalizeRoute(url) {
  // Replace UUIDs and numeric IDs to avoid key explosion
  // /users/123/posts/456 -> /users/:id/posts/:id
  return url
    .split('?')[0] // strip query string
    .replace(/\/[0-9a-f]{8}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{12}/gi, '/:uuid')
    .replace(/\/\d+/g, '/:id');
}

module.exports = { resolveKey, extractIP, extractAPIKey, extractUserId };
```

### `src/utils/headers.js`

```javascript
'use strict';

/**
 * Sets standard rate limit response headers.
 * Follows the IETF RateLimit header fields draft:
 * https://datatracker.ietf.org/doc/draft-ietf-httpapi-ratelimit-headers/
 */
function setRateLimitHeaders(res, { limit, remaining, resetAtSeconds, retryAfterSeconds }) {
  // Legacy headers (widely supported)
  res.setHeader('X-RateLimit-Limit', limit);
  res.setHeader('X-RateLimit-Remaining', Math.max(0, remaining));
  res.setHeader('X-RateLimit-Reset', resetAtSeconds); // Unix timestamp

  // IETF draft standard headers
  res.setHeader('RateLimit-Limit', limit);
  res.setHeader('RateLimit-Remaining', Math.max(0, remaining));
  res.setHeader('RateLimit-Reset', resetAtSeconds);

  if (retryAfterSeconds !== undefined) {
    // RFC 7231 — required on 429 responses
    res.setHeader('Retry-After', Math.ceil(retryAfterSeconds));
  }
}

function calculateResetTime(currentWindowStart, windowSizeSeconds) {
  // Reset at the END of the current window
  return currentWindowStart + windowSizeSeconds;
}

function calculateRetryAfter(currentWindowStart, windowSizeSeconds, nowMs) {
  const resetAt = (currentWindowStart + windowSizeSeconds) * 1000;
  return Math.max(0, (resetAt - nowMs) / 1000);
}

module.exports = { setRateLimitHeaders, calculateResetTime, calculateRetryAfter };
```

### `src/config/rules.js`

```javascript
'use strict';

/**
 * Rate limit rule configuration.
 *
 * Rules are evaluated in order — first match wins.
 * This allows tiered limits: authenticated > unauthenticated,
 * premium > standard, etc.
 */

const defaultRules = {
  // Global defaults
  global: {
    windowSizeSeconds: 60,
    limit: 100,
    keyBy: 'auto',
    perRoute: false,
  },

  // Route-specific overrides
  routes: [
    {
      // Auth endpoints — stricter to prevent credential stuffing
      match: /^\/auth\//,
      windowSizeSeconds: 900, // 15 minutes
      limit: 10,
      keyBy: 'ip',
      perRoute: true,
    },
    {
      // Password reset — very strict
      match: /^\/auth\/password-reset/,
      windowSizeSeconds: 3600, // 1 hour
      limit: 5,
      keyBy: 'ip',
      perRoute: true,
    },
    {
      // Public search — moderate
      match: /^\/search/,
      windowSizeSeconds: 60,
      limit: 30,
      keyBy: 'auto',
      perRoute: true,
    },
    {
      // Webhook endpoints — high volume expected
      match: /^\/webhooks\//,
      windowSizeSeconds: 60,
      limit: 1000,
      keyBy: 'ip',
      perRoute: false,
    },
  ],

  // Tier-based overrides (applied after route matching)
  // Checked via req.rateLimitTier set by auth middleware
  tiers: {
    premium: { multiplier: 10 },
    standard: { multiplier: 1 },
    free: { multiplier: 0.5 }, // Stricter for free tier
  },
};

function resolveRule(req, config = defaultRules) {
  const path = req.path || req.url?.split('?')[0] || '/';

  // Find matching route rule
  let rule = { ...config.global };

  for (const routeRule of config.routes || []) {
    if (routeRule.match.test(path)) {
      rule = { ...rule, ...routeRule };
      break; // First match wins
    }
  }

  // Apply tier multiplier if present
  const tier = req.rateLimitTier || 'standard';
  const tierConfig = config.tiers?.[tier];
  if (tierConfig?.multiplier) {
    rule = {
      ...rule,
      limit: Math.floor(rule.limit * tierConfig.multiplier),
    };
  }

  return rule;
}

module.exports = { defaultRules, resolveRule };

### `src/middleware/rateLimiter.js`

```javascript
'use strict';

const RedisStore = require('../store/redisStore');
const InMemoryFallback = require('../algorithms/inMemoryFallback');
const { resolveKey } = require('../utils/keyResolver');
const { setRateLimitHeaders, calculateResetTime, calculateRetryAfter } = require('../utils/headers');
const { resolveRule } = require('../config/rules');

/**
 * Factory function — creates a rate limiter middleware instance.
 *
 * @param {Object} options
 * @param {RedisStore} options.store - Redis store instance (required)
 * @param {Object} options.config - Rule configuration (optional, uses defaults)
 * @param {string} options.fallbackBehavior - 'allow' | 'deny' | 'memory' (default: 'memory')
 * @param {Function} options.onLimitReached - Called when a request is denied
 * @param {Function} options.onError - Called on internal errors
 * @param {boolean} options.dryRun - Log but never actually block (default: false)
 * @param {boolean} options.skipSuccessfulRequests - Only count failed requests (default: false)
 */
function createRateLimiter(options = {}) {
  const {
    store,
    config,
    fallbackBehavior = 'memory',
    onLimitReached,
    onError,
    dryRun = false,
    keyPrefix = 'rl',
  } = options;

  if (!store) {
    throw new Error('[RateLimiter] options.store is required');
  }

  const fallback = fallbackBehavior === 'memory' ? new InMemoryFallback() : null;

  return async function rateLimiterMiddleware(req, res, next) {
    const nowMs = Date.now();

    try {
      // 1. Resolve which rule applies to this request
      const rule = resolveRule(req, config);

      // 2. Resolve the rate limit key
      const { key, identifierType } = resolveKey(req, {
        keyPrefix,
        keyBy: rule.keyBy,
        perRoute: rule.perRoute,
        keyGenerator: rule.keyGenerator,
      });

      // 3. Execute rate limit check
      let result = await store.slidingWindowIncrement({
        key,
        windowSizeSeconds: rule.windowSizeSeconds,
        limit: rule.limit,
        nowMs,
      });

      // 4. Handle Redis unavailability
      if (result === null) {
        result = await handleFallback({
          fallbackBehavior,
          fallback,
          key,
          rule,
          nowMs,
          onError,
        });
      }

      // 5. Calculate header values
      const resetAt = calculateResetTime(result.currentWindowStart, result.windowSizeSeconds);
      const remaining = Math.max(0, rule.limit - result.weightedCount);
      const retryAfter = result.allowed
        ? undefined
        : calculateRetryAfter(result.currentWindowStart, result.windowSizeSeconds, nowMs);

      // 6. Set response headers (always, even on denied requests)
      setRateLimitHeaders(res, {
        limit: rule.limit,
        remaining,
        resetAtSeconds: resetAt,
        retryAfterSeconds: retryAfter,
      });

      // 7. Attach rate limit info to request for downstream use
      req.rateLimit = {
        limit: rule.limit,
        remaining,
        resetAt,
        identifierType,
        key,
        allowed: result.allowed,
        weightedCount: result.weightedCount,
      };

      // 8. Allow or deny
      if (!result.allowed && !dryRun) {
        if (typeof onLimitReached === 'function') {
          onLimitReached(req, res, rule, result);
          return; // onLimitReached is responsible for sending the response
        }

        return res.status(429).json({
          error: 'Too Many Requests',
          message: `Rate limit exceeded. Try again in ${Math.ceil(retryAfter)} seconds.`,
          retryAfter: Math.ceil(retryAfter),
          limit: rule.limit,
          resetAt,
        });
      }

      if (!result.allowed && dryRun) {
        console.warn(`[RateLimiter] DRY RUN: Would have blocked ${key}`);
      }

      next();
    } catch (err) {
      console.error('[RateLimiter] Unexpected error:', err);

      if (typeof onError === 'function') {
        return onError(err, req, res, next);
      }

      // Default: fail open — don't block requests due to rate limiter bugs
      // OPERATIONAL NOTE: This is a deliberate choice. Change to fail-closed
      // if your threat model requires it.
      console.warn('[RateLimiter] Failing open due to internal error');
      next();
    }
  };
}

async function handleFallback({ fallbackBehavior, fallback, key, rule, nowMs, onError }) {
  switch (fallbackBehavior) {
    case 'allow':
      // Fail open — let all requests through
      return {
        allowed: true,
        weightedCount: 0,
        currentCount: 0,
        previousCount: 0,
        currentWindowStart: Math.floor(nowMs / 1000) - (Math.floor(nowMs / 1000) % rule.windowSizeSeconds),
