# Gemini — Synthesis Challenge

**Fatal objection:** The experiment's near-ceiling performance (A: 9/10, B: 9/10) combined with highly directive vocabulary ("zombie-write failure modes", "fencing tokens") in both variants prevents observing any differential impact of slot placement on detection rate, thus weakening the conclusion that slot is irrelevant for vocabulary-driven orientation.

**Why it matters:** If slot placement does affect reasoning when the vocabulary is less directive or the task is more challenging, the current conclusion prematurely dismisses a key mechanism for installing consideration sets.

**What resolves it:** A redesigned exp-21 that compares vocabulary in different slots and in the artifact itself on a demonstrably non-ceiling task, using less directive vocabulary that does not explicitly name both the failure mode and the fix.
