# Exp-23 CC Decision

**Date:** 2026-03-30

## Objection Status: Accepted (Significant but not fatal)

The pattern-matching confound is accepted as a significant qualification but not fatal to the
primary finding. H2 remains robustly corroborated across four independent experiments
(exp-19, exp-20, exp-21b, exp-23); pattern-matching vs search is a mechanistic interpretation
question that does not invalidate the slot-irrelevance finding.

The P_p simulation engine / P_d checklist handler interpretation is held as tentative pending
follow-up.

## Conclusion (Narrowed)

**Established:**
1. Mechanism vocabulary in either slot drives ceiling detection (A=10/10, B=10/10, C=0/10).
2. H2 further corroborated: content operative regardless of slot.
3. Mechanism vocabulary (naming the causal chain) is above the specificity threshold.
4. The specificity threshold lies between orientation level (0/10, exp-21b) and mechanism
   level (10/10, exp-23) — both ceiling/floor.

**Not established:**
- Exact specificity threshold location within the orientation→mechanism range.
- Whether P_p "simulation engine" vs P_d "checklist handler" is structural or a verbosity artifact.
- Whether mechanism detection is discovery (consideration-set activation) or pattern recognition.

## What Remains from This Experiment

The token count reversal (A=1166 > B=1009 for mechanism vocab, vs B > A for label vocab in
exp-20) is an observation requiring replication. If stable across models and runs, it supports
different processing modes for the two slot types.
