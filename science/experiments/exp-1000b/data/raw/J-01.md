Two engineers. Same model. Same ten-year label. One gets a competent review. The other gets the race condition that only fires under 50ms contention windows.

I ran twelve experiments to find out why.

The answer isn't what most practitioners expect. It's not how forcefully you encode the behavior. "Cannot help but catch every bug" sounds powerful. I know how that sentence ends. Strip the domain vocabulary, keep the compulsion — output collapses to baseline. Strip the compulsion, keep the domain vocabulary — the outputs hold. The compulsion framing was always arriving with dense domain vocabulary underneath it, taking credit for work it didn't do.

What actually shapes the consideration set is the technical substrate adjacent to the identity anchor. The range of failure modes the model will even reach for. Not the instruction. The neighborhood.

Not "senior software engineer."

That's orientation vocabulary. It labels a category without entering it. The model generates things consistent with senior-engineer-ness without running deep in any particular direction — no friction, no resistance, outputs sliding smooth and shallow. The credential points at a room. It doesn't go inside.

Mechanism vocabulary goes inside.

"Whether the write is fenced. Whether the lock is held for the duration of the operation. Whether the TTL arithmetic holds under GC pause."

Those tokens name the causal chain. They pre-weight the model's attention toward the failure-mode cluster before the first task token is processed — like loading a dirty sector map before the read head moves, the consideration set is already shaped before the task is parsed. You feel it in the output before you can name why. The pull is already there.

Orientation vocabulary: 0/10 detection on target findings. Mechanism vocabulary: 10/10.

That gap doesn't close with effort. It's structural. Twelve experiments, same result. The mechanism doesn't care how much you want it to behave differently.

The practical rule is simpler than the mechanism: write the technical substrate, not the credential.

Every orientation phrase has a mechanism equivalent. One tells the model where to look. The other tells it what a torn write looks like when it gets there.
